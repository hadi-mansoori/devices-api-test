
## Device API Service
### Written by Sayed Hadi Mansoori Rad

## Installation

```bash
$ yarn install
```

## Running migrations

```bash
# develop mode
$ yarn migration:run
```

## Running the app

```bash
# develop mode
$ yarn run start:dev
```

```bash
# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test
```

```bash
# e2e tests
$ yarn run test:e2e
```

```bash
# test coverage
$ yarn run test:cov
```
## Migration commands

```bash
# create migration in windows
$ yarn migration:create:windows 

# create migration in linux
$ yarn migration:create:linux 

# revert migrations
$ yarn migration:revert
```
