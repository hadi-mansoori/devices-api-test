## Installation

```bash
$ yarn install
```

## Running the app

```bash
# develop mode
$ yarn run start:dev
```

```bash
# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test
```

```bash
# e2e tests
$ yarn run test:e2e
```

```bash
# test coverage
$ yarn run test:cov
```
